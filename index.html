    Welcome to Morph Community.

    Ok lets get started. If you are a hardware developer you will need to download Eagle Cad if you don't already have it. And if you are a software developer you will need to download Code Composure Studio (CSS) for texas instruments.

    Once you have download the appropriate software the next step is to clone the repository:

    $ cd your_local_directory/your_repo_name
$ git clone https://github.com/researcher111/reconPlat.git

    Hardware Designer

    Once you have cloned the repo. You can navigate to the folder called hardware it hard the collection of hardware files that you will to open in eagle.

    Software developers

    Once you have clone the repo the is a Code Composure Studio project in the software folder. If you open this project it will open up the Code Composure Studio with the right environmental configurations.
